EarAid

Web audio player for music learning.

Who needs EarAid

EarAid is built for those musicians who need a clean audio player with some helpful tools for transcribing, learning, and playing along with musical pieces.

How to start

First of all click on the "browse" icon and select an audio file to upload, once uploaded it will be automatically visualized as a waveform. You can now play it and navigate trough it with the cursor just as you would do on any DAW. There's also a green minimap under the main wave wich always displays the whole waveform, it's useful when zoom is applied.

Tools overview

Markers

To crceate a marker press the flag icon and name the new marker in the window that will appear. To remove it click on the undo icon or double click on it if you are on pc.

Selection

Marking up a selection is needed either to loop a fragment of your audiofile or to mark the fragment you want to be cropped with the download tool. To set ou update the selection start position put the cursor on the right point of the waveform and click the icon , same thing for the end point icon . To clear up the selection click on . To activate/turn off selection looping click on the loop icon , it will turn red when activated.

Spectrogram view

To toggle spectrogram visualization click on the icon , consider that the app takes some time (some seconds, more if the file is long) to render the spectrogram every time you turn it on, change the window size or change the zoom parameter.

Freeze

EarAid has a granular freeze tool wich aims to freeze the spectrum of a small time window (100ms) around the cursor. To turn it on/off click on the icon . This could be helpful if you need to figure out a chord or some holded parts. Note that it freeze everything is in the time window so could not work well in the part you want to hear is not well isolated.

Download

The download feature applyes the time stretch and pitch shift parameters to the audio files, crops the selection if active, encodes the modified audio in a .wav file and serves it as a download. Click on the icon to do so.

Parameters overview

Zoom

You can change zoom level from 0% (the whole waveform is visualized) to 100% (200 pixels per second).

Pitch Shift

The pitchshift parameter can be used to adjust the audio pitch, in those cases you need to hear the piece in a different key. You can also click on the icon to reset. It's made with an implementation of soundtouch.js audio worklet.

Speed

The speed parameter can be used to slow down or speed up the playback without affecting pitch. You can also click on the icon to reset. It is made using the HTMLMediaElement playbackSpeed feature.

Volume

Adjust the master volume. You can also click on the icon to mute/unmute.

Piano

EarAid also features an html piano keyboard wich could be useful to check what you hear. The sound is the sawtooth default synth by Tone.js.

Key shortcuts

Spacebar Play/Pause

R Resume audio playback.
F Adds a marker.
S Sets selection starting point.
E Sets selection ending point.
L Toggles loop.
W Rewinds.
Delete Clear selection.

Left/Right Arrows -/+ 5 seconds
+/- Adjust zoom.
P Toggles piano mode.
1->7 When piano mode is on selects playng octave.
A->K When piano mode is on plays the piano.

Compatibility issues

Earaid performs best on Windows, Linux, and Android systems with Edge, Chrome, and Firefox browsers. On macOS, it is compatible with Chrome and Firefox but may encounter issues with Safari. On iOS and iPadOS, it has some problems that appear to stem from the different implementation of the new WebAudioWorklet system.
