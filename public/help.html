EarAid

Web audio player for music learning.

Who needs EarAid

EarAid is built for those musicians who need a clean audio player with some helpful tools for transcribing, learning, and playing along with musical pieces.

How to start

First of all, click on the "browse" icon and select an audio file to upload. Once uploaded, it will be automatically visualized as a waveform. You can now play it and navigate through it with the cursor just as you would do on any DAW. There's also a green minimap under the main wave which always displays the whole waveform; it's useful when zoom is applied.

Tools overview

Markers

To create a marker, press the flag icon and name the new marker in the window that will appear. To remove it, click on the undo icon or double-click on it if you are on PC.

Selection

Marking up a selection is needed either to loop a fragment of your audio file or to mark the fragment you want to crop with the download tool. To set or update the selection start position, put the cursor on the right point of the waveform and click the icon , and do the same for the end-point icon . To clear the selection, click on . To activate/turn off selection looping, click on the loop icon ; it will turn red when activated.

Spectrogram view

To toggle spectrogram visualization, click on the icon . Consider that the app takes some time (a few seconds, more if the file is long) to render the spectrogram every time you turn it on, change the window size, or change the zoom parameter.

Freeze

EarAid has a granular freeze tool which aims to freeze the spectrum of a small time window (100ms) around the cursor. To turn it on/off, click on the icon . This could be helpful if you need to figure out a chord or some held parts. Note that it freezes everything in the time window, so it might not work well if the part you want to hear is not well isolated.

Download

The download feature applies the time stretch and pitch shift parameters to the audio files, crops the selection if active, encodes the modified audio in a .wav file, and serves it as a download. Click on the icon to do so.

Parameters overview

Zoom

You can change the zoom level from 0% (the whole waveform is visualized) to 100% (200 pixels per second).

FFT

It is possible to set the fft window width or turn the fft visualization off. A wider window will provide more definition on the lower tones, but with a thinner one but with a thinner one you will achieve greater precision in the timing of events.

Pitch Shift

The pitch shift parameter can be used to adjust the audio pitch, in cases where you need to hear the piece in a different key. You can also click on the icon to reset. It's made with an implementation of soundtouch.js audio worklet.

Speed

The speed parameter can be used to slow down or speed up the playback without affecting pitch. You can also click on the icon to reset. It is made using the HTMLMediaElement playbackSpeed feature.

Volume

Adjust the master volume. You can also click on the icon to mute/unmute.

Piano and FFT

EarAid also features an HTML piano keyboard which could be useful to check what you hear. The sound is a sawtooth synth by Tone.js. Below the keyboard, you can activate an fft note guessing visualization.

Key shortcuts

F Add a marker.

L Toggle loop.
W Rewind.

Left/Right Arrows -/+ 5 seconds

P Toggle piano mode.
1->7 When piano mode is on selects playing octave.
A->K When piano mode is on plays the piano.

Compatibility issues

EarAid performs best on Windows, Linux, and Android systems with Chrome, Edge, and Firefox browsers. On macOS, it is compatible with Chrome and Firefox but may encounter issues with Safari. On iOS and iPadOS, it has some problems that appear to stem from the different audio api implementation in webkit.
